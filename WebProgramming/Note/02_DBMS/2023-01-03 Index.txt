Index: SQL 명령문의 처리 속도를 향상시키기 위해서 컬럼에 대해서 생성하는 오라클 객체
	- Index는 B* 트리 형식으로 구성
	- Index를 설정하면 트리도 생성해야 하기 때문에 추가적인 시간과 공간 필요
	- Index로 사용된 컬럼 값이 변경되어도 트리 구조 역시 수행
		- 이 작업은 오라클 서버가 자동으로 함으로 생략 불가능, 
		   따라서 Index가 있는 경우 DML 작업이 훨씬 무거워짐.

Index 생성
CREATE INDEX 인덱스이름 ON 테이블(항목);

Index 제거
DROP INDEX 인덱스이름;

Index를 사용하면 좋은 경우
	1. 테이블에 전체 행의 수가 100000건 이상일때
 	2. 위의 쿼리문을 전체 쿼리문 들 중에서 95% 사용
 	3. 쿼리문의 결과로 구해지는 행은 100건 정도
